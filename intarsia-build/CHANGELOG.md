## [0.1.0-alpha] - 2026-03-04

### Added

- Initial alpha release of intarsia-build
- `compile_isle_auto()` - Automatically discover and compile ISLE files in `isle/` directory
- `compile_isle_dir()` - Compile all ISLE files in a specified directory
- `compile_isle_file()` - Compile a single ISLE file to Rust code
- `compile_isle_files()` - Compile multiple specific ISLE files
- Build script helpers for compiling ISLE DSL to Rust using cranelift-isle
